vite是什么？

vite是面向现代浏览器，基于原生模块系统ESModule实现了按需编译的Web开发构建工具，在生产环境下基于Rollup打包。但目前vite并不太成熟，在编译打包阶段可能会造成文件丢失，可以先用比较成熟的webpack进行编译打包。
